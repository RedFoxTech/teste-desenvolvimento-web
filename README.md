## Primeiro

```bash
npm install 
# ou
yarn install 
```
depois, rode o servidor de desenvolvimento:

```bash 
npm run dev
# or
yarn dev
```
How, to run the production server:
```bash 
npm run build
# or
yarn build
```
Shortly after run 
```bash 
npm run start
# or
yarn start
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
